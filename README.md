# 🚀 Laiter Messenger

**Laiter** — современный мессенджер с NFC-подарками, биржей заданий и NFT-магазином.

## ✨ Возможности

### 💬 Чат
- Личные сообщения
- Глобальный чат
- Системные чаты (Аукцион, Розыгрыши, Ивенты)
- Онлайн-статус
- Отправка фото

### 🎁 NFC Магазин
- 11 NFT-подарков
- Покупка за Лайтерки
- Активация в инвентаре

### 📋 Биржа заданий
- Создание заданий (только верифицированные)
- Принятие заданий
- Одобрение/отклонение
- Автоперевод монет

### 👑 Админ-панель
- Управление пользователями
- Выдача монет и галочек
- Сброс данных

## 🛠 Технологии

| Компонент | Технология |
|-----------|------------|
| Backend | Node.js + Express |
| Real-time | Socket.IO |
| Хранение | JSON файлы |
| Фронтенд | HTML5, CSS3, JS |
